Главная страница находится в файле index.html
Страница с товаром в файле index2.html

Скрипты, которые загружают информацию для блоков на главной странице находятся в папке js/ui/modules

Запросы к серверу организованы через систему реквестов - js/requests/request
Там можно ознакомиться с примерами
Например js/requests/product/filteredlist Запрос на http://api.twiger.mcdir.ru/ по пути product/filteredlist.
Этот запрос используется в js\ui\modules\recommendation_block.js

У класса модуля есть параметр anchor, он указывает в какой блок будет добавлен этот класс.
Сделано так, по причине того что пока с сервера не приходит список модулей, которые можно было бы делать на стороне бекэнда.
