ТЗ по сайту с подарками.

Что необходимо сделать.
Главную страницу и выдачу товара.
Так же есть в недоработанном состоянии страница Товара.

Главная страница состоит из модулей. Предполагается, что каждый модуль будет подгружен с сервера. 
Модуль представляет собой отдельный класс написанный на JS и отнаследованный от корневого класса, который представляет модули.

Список модулей главной страницы.
 - Хедер (заготовка)
 - Поиск, с возможностью настройки фильтрации (переводит на страницу "выдача товара" (заготовка, поиск не работает)
 - Модуль в который загружены 3 товара ( По определенному признаку, по какому нужно будет уточнить) (заготовка)
 - Модуль "Могут понравиться" Выдает контекстуальные товары (готово)
 - Модуль "Самые просматривыемые" выдает список товаров, у которых больше всего просмотров (готово )
 - Рекламный блок (Готово )
 - Модуль "Лучшие магазины недели" (Заготовка)
 - Модуль "Важно знать" (Заготовка)
 - Футер (Заготовка)
 
 
Список модулей страницы "Выдача товара" (Страницы нет)
 - Хедер (можно взять с главной)
 - Поиск (Так же заготовка есть на главной)
 - Список товаров (состоит из двух частей)
    - Левая часть(левая): 
      - представляет собой список товаров, в 3 блока в ширину и бесконечно по высоте
      - каждые 4-5 блоков, рекламный модуль по ширине 3-ех блоков.
    - вторая часть (правая)
      - модуль фильтров
 - Футер

Панель управления сайтом для наполнения серверной части: http://api.twigger.mcdir.ru/admin/
